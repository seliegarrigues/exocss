Bienvenue dans le monde du CSS

CSS, ou Cascading Style Sheets, est un langage de style utilisé pour décrire la présentation d'un document écrit en HTML ou XML. CSS permet de contrôler la mise en page, les couleurs, les polices et les animations des éléments d'une page web.
